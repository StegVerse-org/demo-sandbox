# StegVerse Demo Sandbox

Ephemeral sandbox for GCAT/BCAT formalism exploration.

## Structure

```
invariants/      — GCAT (I1-I6) and BCAT (B1-B6) checks
sandbox/         — Ephemeral experiment runner
experiments/     — Pre-built formalism suites
```

## Quick Start

```bash
pip install -r requirements.txt
python -m experiments.experiment_suite
```

## Integration

Sandbox exports experimental data for formalism refinement.
